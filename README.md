# Typescript

## git command

新しいブランチ作成

```
git checkout -b BranchName
```

状態確認

```
git status
```

修正した内容をステージに追加

```
git add .
```

変更をローカルリポジトリにコミット

```
git commit -m "message"
```

リモートリポジトリに変更反映

```
git push -u origin HEAD
```

メインブランチに移動

```
git checkout -
```

ブランチをメインにマージする

```
git merge -
```

マージしたメインブランチをリモートにも反映

```
git push origin HEAD
```
